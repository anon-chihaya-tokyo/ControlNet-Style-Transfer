# Roadmap

## v0.2

- Add batch generation support
- Add optional prompt templates for portrait, anime, product, and landscape scenes
- Add image size presets in the UI
- Save prompt and inference metadata alongside output images

## v0.3

- Support multiple ControlNet preprocessors such as depth, lineart, and softedge
- Add optional real ESRGAN or latent upscaling workflow
- Add before/after comparison gallery in Gradio
- Add queue support for multi-user inference sessions

## v0.4

- Add configuration via `.env` or YAML
- Add basic test coverage for image preprocessing and prompt assembly
- Add structured logging and error reporting
- Add Docker deployment example

## Long-term

- Better style extraction than simple color and brightness heuristics
- LoRA loading and switching from the UI
- Model preset management
- REST API mode for integration with external apps
